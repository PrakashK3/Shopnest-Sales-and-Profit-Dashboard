# Shopnest-Sales-and-Profit-Dashboard
An interactive Sales & Profit Dashboard built using Microsoft Excel to analyze business performance across categories, customers, time, and geography. This dashboard transforms raw sales data into meaningful insights for decision-making.
🚀 Project Highlights
💰 Total Sales: $1.9M+
📈 Total Profit: $247K+
📊 Dynamic and interactive visualizations
🎯 Business-focused insights for performance tracking
📌 Key Features
🔹 KPI Metrics
Total Sales
Total Profit
🔹 Interactive Filters (Slicers)
Category: Furniture, Office Supplies, Technology
Year: 2021 – 2024
Month: Jan – Dec

📊 Profit by Category (Year-wise comparison)
📦 Sales by Sub-Category
👥 Top 5 Customers
🌎 Sales by State (Geo Map)

🖨️ Copiers & Phones dominate sub-category sales
